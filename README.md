# 🧠 Handwritten Digit Recognition using MNIST

This project demonstrates a simple implementation of a neural network for recognizing handwritten digits using the MNIST dataset. Built with TensorFlow and Keras, it is beginner-friendly and runs easily on Google Colab.

---

## 📌 Features

- Uses the MNIST dataset (28x28 grayscale images)
- Fully connected neural network (Dense layers)
- Data normalization for better performance
- Model training and validation
- Prediction on test data
- Visualization of sample digit

---

## 🛠️ Tech Stack

- Python
- TensorFlow / Keras
- NumPy
- Matplotlib
